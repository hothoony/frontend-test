position values:
    - static (기본 값)
        - top, bottom, left, right 값을 적용받지 않는다
        - 페이지 흐름에 따라 배치된다
    - relative
        - top, bottom, left, right 값을 적용받는다
    - fixed
        - viewport 기준으로 배치된다
        - 페이지가 스크롤되도 같은 위치에 배치된다
        - 스크롤 상관없이 화면 기준으로 배치
    - absolute
    - sticky
box1 
    .box1 {
        /* position: static; */
        top: 50px;
        left: 50px;
    }
box2 
    .box2 {
        position: fixed;
        top: 50px;
        right: 50px;
    }
box3 
    .box3 {
        /* parent 가 'position: relative;' 이어야 함 */
        position: absolute;
        top: 150px;
        right: 150px;
    }
box4 
    .box4 {
        position: relative;
        top: 150px;
        right: 150px;
    }
